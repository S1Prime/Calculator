# Calculator
This is a python code for a Calculator
Simple Calculator using Python Tkinter

This project is a basic GUI Calculator built using Python and the Tkinter library. It performs simple arithmetic operations like:

Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)

The calculator provides a simple and user-friendly interface where users can click buttons to enter numbers and operators, then press = to get the result.

#Features:
Simple GUI using Tkinter
Supports basic arithmetic operations
Error handling for invalid expressions
Lightweight and beginner-friendly project

#Technologies Used:
Python 3
Tkinter (Python GUI Library)
How It Works
User enters numbers and operators using the calculator buttons.
The entered expression is displayed on the screen.
When = is pressed, the program uses Python’s eval() function to calculate the result.
If the expression is invalid, it displays Error.
